## To create a new user in Acquirer use the following steps
1. Log in to the ```Acquirer UI``` and navigate to:
     - ```System → Security → Users → New```
     ![step 1](/img/Picture21.png "step 1")
2. Add ```user information``` and set a ```password```.
![step 2](/img/Picture22.png "step 2")
3. Add the ```user``` to a ```group```.
![step 3](/img/Picture23.png "step 3")
