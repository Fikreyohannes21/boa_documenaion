# Introduction to BASE24-eps
```BASE24-eps``` is a payment processing engine that provides a full range of functionality to support a variety of
payment transactions. ```BASE24-eps``` supports all leading payment cards and interfaces to leading devices, switches, and back-end host systems. ACI’s fault-tolerant application software takes advantage of the best in systems software for reliability, availability, scalability, and high-performance transaction throughput.


### BASE24-eps provides the following features:
- Multiple platform and operating system support (for example, AIX on IBM System p server, Red Hat Enterprise Linux (RHEL) on any x86-64 server, Guardian on HP Integrity NonStop or HP Integrity NonStop BladeSystem servers).
- Multiple database support (that is, c-tree, Enscribe, Postgres and Oracle).
- Scriptable online transaction authentication, authorization, and journal perusal/reporting/extract.
- Flexible routing and switching including primary, two alternate, and default routing destinations for prescreening,
authorization, advices, and impacting.
- Triple DES cryptographic support for transaction security using tamper-proof, hardware security modules
(HSMs) from five different vendors.
- Graphical user interface with user security and user auditing capabilities validated against payment industry best
practices such that customers can become PCI-compliant.
- Integration with multiple ACI Enterprise Payment Solution products.
- EMV support
- Multi-currency support
- Full-file replacement or partial record updates for data source refreshing.
- Object-oriented, component-based, layered architecture authored in the C++ and Java™ software development
languages.
